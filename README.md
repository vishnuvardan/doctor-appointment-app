# Doctor Appointment App

This repository is structured as a monorepo containing the frontend and backend of the Doctor Appointment Application.

## Directory Structure

- `frontend/`: Contains the frontend web application (Angular).
- `backend/`: Folder dedicated to backend development.
- `.vscode/`: Contains shared editor settings, launch configurations, and build/test tasks.

## Getting Started

### Frontend
Navigate to the `frontend/` folder to install dependencies and run the development server:
```bash
cd frontend
npm install
npm start
```
